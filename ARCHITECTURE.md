# 系统架构文档

## 系统概览

本项目是一个基于STM32F103C8T6的四轮小车控制系统，采用模块化设计，易于扩展和维护。

## 模块说明

### 1. 主控制模块 (main.c)

**职责：**
- 系统初始化
- 主循环控制
- PS2输入处理
- 运动控制决策

**主要函数：**
```c
int main(void)                              // 主函数
void SystemClock_Config(void)               // 时钟配置
void MX_GPIO_Init(void)                     // GPIO初始化
void MX_SPI1_Init(void)                     // SPI初始化
void MX_TIM2_Init(void)                     // 定时器2初始化
void MX_TIM3_Init(void)                     // 定时器3初始化
void ProcessPS2Input(PS2_Data_t* ps2_data)  // PS2输入处理
```

**控制流程：**
```
开始
  ↓
系统初始化
  ↓
外设初始化
  ↓
启动PWM
  ↓
初始化驱动
  ↓
┌─────────────┐
│  主循环     │
│             │
│  读取PS2    │ ←─┐
│     ↓       │   │
│  处理输入   │   │
│     ↓       │   │
│  控制电机   │   │
│     ↓       │   │
│  延时20ms   │   │
│     └───────┼───┘
└─────────────┘
```

### 2. L298N驱动模块

**文件：** `Middlewares/L298N/l298n.c/h`

**职责：**
- 电机方向控制
- 电机速度控制（PWM）
- 运动模式控制（前进、后退、转向）

**API函数：**
```c
void L298N_Init(void);                              // 初始化
void L298N_SetSpeed(Motor_t motor, uint16_t speed); // 设置速度
void L298N_SetDirection(Motor_t motor, MotorDir_t direction); // 设置方向
void L298N_MoveForward(uint16_t speed);             // 前进
void L298N_MoveBackward(uint16_t speed);            // 后退
void L298N_TurnLeft(uint16_t speed);                // 左转
void L298N_TurnRight(uint16_t speed);               // 右转
void L298N_Stop(void);                              // 停止
```

**数据结构：**
```c
typedef enum {
    MOTOR_FL = 0,  // 前左
    MOTOR_FR = 1,  // 前右
    MOTOR_RL = 2,  // 后左
    MOTOR_RR = 3,  // 后右
} Motor_t;

typedef enum {
    DIR_STOP = 0,      // 停止
    DIR_FORWARD = 1,   // 前进
    DIR_BACKWARD = 2   // 后退
} MotorDir_t;
```

### 3. PS2驱动模块

**文件：** `Middlewares/PS2/ps2.c/h`

**职责：**
- PS2手柄通信
- 按键状态读取
- 模拟摇杆数据读取

**API函数：**
```c
void PS2_Init(void);                                        // 初始化
void PS2_ReadData(PS2_Data_t* data);                       // 读取数据
uint8_t PS2_IsButtonPressed(PS2_Data_t* data, uint16_t button); // 检查按键
```

**数据结构：**
```c
typedef struct {
    uint16_t buttons;   // 按键状态（位图）
    uint8_t right_x;    // 右摇杆X轴 (0-255)
    uint8_t right_y;    // 右摇杆Y轴 (0-255)
    uint8_t left_x;     // 左摇杆X轴 (0-255)
    uint8_t left_y;     // 左摇杆Y轴 (0-255)
} PS2_Data_t;
```

**按键定义：**
```c
PS2_BTN_UP, PS2_BTN_DOWN, PS2_BTN_LEFT, PS2_BTN_RIGHT
PS2_BTN_TRIANGLE, PS2_BTN_CIRCLE, PS2_BTN_CROSS, PS2_BTN_SQUARE
PS2_BTN_L1, PS2_BTN_L2, PS2_BTN_L3
PS2_BTN_R1, PS2_BTN_R2, PS2_BTN_R3
PS2_BTN_SELECT, PS2_BTN_START
```

## 数据流

### 控制数据流
```
PS2手柄
   ↓ (无线)
PS2接收器
   ↓ (SPI)
STM32读取按键/摇杆数据
   ↓
解析和处理输入
   ↓
计算运动方向和速度
   ↓
设置各电机参数
   ↓ (GPIO + PWM)
L298N驱动
   ↓
电机运转
```

## 中断和定时器

### 使用的定时器

| 定时器 | 用途 | 频率 | 通道 |
|--------|------|------|------|
| TIM2   | PWM输出（前轮） | 1kHz | CH1, CH2 |
| TIM3   | PWM输出（后轮） | 1kHz | CH3, CH4 |
| SysTick| 系统时基 | 1kHz | - |

### PWM配置
- **时钟源**：72MHz (系统时钟)
- **预分频**：71 (72MHz / 72 = 1MHz)
- **自动重装载值**：999 (1MHz / 1000 = 1kHz)
- **占空比范围**：0-1000 (对应0%-100%)

## 内存使用

### Flash (64KB)
- 程序代码：~10KB
- HAL库：~20KB (如使用完整HAL)
- 剩余空间：可用于扩展功能

### RAM (20KB)
- 全局变量：~1KB
- 堆栈：4KB
- 剩余空间：可用于缓冲区和额外数据

## 扩展性设计

### 预留扩展点

1. **传感器接口**
   - UART：可连接蓝牙、GPS等
   - I2C：可连接OLED、IMU等
   - ADC：可连接电池监测、距离传感器等

2. **附加功能**
   - 超声波避障（HC-SR04）
   - 循迹传感器（红外）
   - 舵机控制（PWM）
   - LED灯光控制

3. **软件扩展**
   - PID控制算法
   - 路径规划
   - 自动驾驶模式
   - 遥测数据记录

## 性能指标

| 指标 | 值 |
|-----|-----|
| 主频 | 72MHz |

## 调试接口

### SWD接口
- SWDIO: PA13
- SWCLK: PA14
- 用于程序下载和调试

### 串口（预留）
- USART1: PA9(TX), PA10(RX)
- 可用于调试输出

## 安全特性

1. **看门狗（可选）**
   - 防止程序死锁

2. **信号丢失保护**
   - PS2信号丢失时自动停车

3. **过流保护**
   - 硬件：L298N内置保护
   - 软件：可添加电流监测

4. **低电压保护（可选）**
   - ADC监测电池电压
   - 低电压报警
