# 硬件连接详细说明

## 概述
本文档详细说明了STM32F103C8T6小车控制系统的硬件连接方法。

## 完整引脚映射表

### STM32F103C8T6 引脚分配

| STM32引脚 | 功能 | 外设/模块 | 说明 |
|----------|------|----------|------|
| PA0 | GPIO Output | L298N #1 IN1 | 左前轮方向控制 |
| PA1 | GPIO Output | L298N #1 IN2 | 左前轮方向控制 |
| PA2 | GPIO Output | L298N #1 IN3 | 右前轮方向控制 |
| PA3 | GPIO Output | L298N #1 IN4 | 右前轮方向控制 |
| PA4 | GPIO Output | PS2 CS | PS2片选信号 |
| PA5 | SPI1_SCK | PS2 CLK | SPI时钟 |
| PA6 | SPI1_MISO | PS2 DAT | SPI数据输入 |
| PA7 | SPI1_MOSI | PS2 CMD | SPI数据输出 |
| PA15 | TIM2_CH1 | L298N #1 ENA | 左前轮PWM |
| PB0 | GPIO Output | L298N #2 IN1 | 左后轮方向控制 |
| PB1 | GPIO Output | L298N #2 IN2 | 左后轮方向控制 |
| PB3 | TIM2_CH2 | L298N #1 ENB | 右前轮PWM |
| PB6 | GPIO Output | L298N #2 IN3 | 右后轮方向控制 |
| PB7 | GPIO Output | L298N #2 IN4 | 右后轮方向控制 |
| PB8 | TIM3_CH3 | L298N #2 ENA | 左后轮PWM |
| PB9 | TIM3_CH4 | L298N #2 ENB | 右后轮PWM |
| 3.3V | 电源 | PS2接收器 | PS2电源 |

| GND | 地 | 全部 | 公共地线 |

## L298N电机驱动模块连接

### L298N #1 (前轮)
```
┌─────────────────────────────────────────┐
│           L298N Motor Driver #1         │
│                                         │
│  IN1 ←──── PA0 (STM32)                 │
│  IN2 ←──── PA1 (STM32)                 │
│  IN3 ←──── PA2 (STM32)                 │
│  IN4 ←──── PA3 (STM32)                 │
│  ENA ←──── PA15 (STM32 TIM2_CH1)       │
│  ENB ←──── PB3 (STM32 TIM2_CH2)        │
│                                         │
│  OUT1 ──→ 左前电机(+)                   │
│  OUT2 ──→ 左前电机(-)                   │
│  OUT3 ──→ 右前电机(+)                   │
│  OUT4 ──→ 右前电机(-)                   │
│                                         │
│  +12V ←─── 电池正极                     │
│  GND  ←─── 公共地(STM32 GND)            │
│  +5V  ──→ (可给STM32供电)               │
└─────────────────────────────────────────┘
```

### L298N #2 (后轮)
```
┌─────────────────────────────────────────┐
│           L298N Motor Driver #2         │
│                                         │
│  IN1 ←──── PB0 (STM32)                 │
│  IN2 ←──── PB1 (STM32)                 │
│  IN3 ←──── PB6 (STM32)                 │
│  IN4 ←──── PB7 (STM32)                 │
│  ENA ←──── PB8 (STM32 TIM3_CH3)        │
│  ENB ←──── PB9 (STM32 TIM3_CH4)        │
│                                         │
│  OUT1 ──→ 左后电机(+)                   │
│  OUT2 ──→ 左后电机(-)                   │
│  OUT3 ──→ 右后电机(+)                   │
│  OUT4 ──→ 右后电机(-)                   │
│                                         │
│  +12V ←─── 电池正极                     │
│  GND  ←─── 公共地(STM32 GND)            │
│  +5V  ──→ (可给STM32供电)               │
└─────────────────────────────────────────┘
```

```
┌───────────────────────────────────┐
│      PS2 Wireless Receiver        │
│                                   │
│  VCC (3.3V) ←──── 3.3V (STM32)   │
│  GND        ←──── GND (STM32)    │
│  DAT (MISO) ←──── PA6 (STM32)    │
│  CMD (MOSI) ←──── PA7 (STM32)    │
│  CLK (SCK)  ←──── PA5 (STM32)    │
│  ATT (CS)   ←──── PA4 (STM32)    │
└───────────────────────────────────┘
```

## 电源系统

### 推荐配置
```
                    ┌────────────────┐
   锂电池(7.4-12V)  │                │
        │           │   L298N #1     │
        ├──────────→│   +12V         │
        │           │                │
        │           └────────────────┘
        │           
        │           ┌────────────────┐
        │           │                │
        ├──────────→│   L298N #2     │
        │           │   +12V         │
        │           │                │
        │           └────┬───────────┘
        │                │
        │           ┌────┴───────────┐
        │           │    +5V 输出    │
        └──────────→│                │
                    │  STM32         │
                    │  (USB或5V输入) │
                    └────────────────┘
                           │
                           │ 3.3V
                           ↓
                    ┌──────────────┐
                    │ PS2接收器    │
                    │   VCC 3.3V   │
                    └──────────────┘
```

### 电源注意事项
1. **电机电源**: 使用7.4V-12V锂电池供电
2. **STM32电源**: 可使用L298N的5V输出或独立5V电源
3. **PS2接收器**: 必须使用3.3V，连接STM32的3.3V输出
4. **共地**: 所有模块的GND必须连接在一起

## 电机连接

### 电机极性
- 如果电机转向与预期相反，交换该电机的两根线即可
- 软件中也可通过修改方向控制逻辑来调整

### 电机布局
```
     前方
  ┌────────┐
  │ FL  FR │  FL=左前轮(Front Left)
  │        │  FR=右前轮(Front Right)
  │ RL  RR │  RL=左后轮(Rear Left)
  └────────┘  RR=右后轮(Rear Right)
     后方
```

## 调试和测试

### 分步测试流程

1. **电源测试**
   - 先不连接电机，只给STM32供电
   - 检查3.3V和5V是否正常
   - 上传程序确认STM32工作正常

2. **PS2通信测试**
   - 连接PS2接收器
   - 配对手柄
   - 通过串口或LED观察是否接收到数据

3. **单个电机测试**
   - 连接一个电机和对应的L298N
   - 测试前进、后退、速度控制
   - 确认后再连接其他电机

4. **完整系统测试**
   - 所有模块连接完成
   - 测试前进、后退、转向
   - 调整速度和方向参数

### 常见问题

1. **电机不转**
   - 检查L298N跳线帽是否安装(ENA/ENB)
   - 检查电机电源
   - 检查PWM信号

2. **电机抖动**
   - 可能是PWM频率不合适
   - 检查电源电压是否稳定
   - 检查负载是否过大

3. **手柄无响应**
   - 检查PS2接收器电源(必须3.3V)
   - 确认手柄已配对
   - 检查SPI接线

4. **方向错误**
   - 交换电机正负极
   - 或修改软件中的方向定义

## 焊接建议

1. 使用杜邦线连接，方便调试
2. 测试稳定后可焊接万能板固定
3. 电源线使用较粗的线(1mm以上)
4. 信号线可使用细导线
5. 预留接口方便维护

## 安全提示

⚠️ **注意事项**
- 调试时先不安装电池，用实验室电源
- 确保电机不会突然启动造成危险
- L298N会发热，必要时加散热片
- 避免短路，特别是电源线
- 首次测试建议架空小车
