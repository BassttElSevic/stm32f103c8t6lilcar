# STM32F103C8T6 小车电控程序

这是一个基于STM32F103C8T6微控制器的四轮小车控制程序，使用PS2手柄进行遥控。

## 硬件需求

### 核心组件
- **1x STM32F103C8T6** 微控制器（Blue Pill开发板）
- **2x L298N** 电机驱动模块
- **4x TT直流电机** (DC motors)
- **1x PS2手柄** 及接收器（通过SPI接口连接）

### 电源
- 电机供电：建议使用7.4V-12V锂电池
- STM32供电：5V（可通过USB或单独电源模块）

## 硬件连接

### L298N电机驱动连接

#### L298N #1 (前轮驱动)
| L298N引脚 | STM32引脚 | 功能说明 |
|----------|----------|---------|
| IN1      | PA0      | 左前轮方向控制1 |
| IN2      | PA1      | 左前轮方向控制2 |
| IN3      | PA2      | 右前轮方向控制1 |
| IN4      | PA3      | 右前轮方向控制2 |
| ENA      | PA15     | 左前轮PWM速度控制 (TIM2_CH1) |
| ENB      | PB3      | 右前轮PWM速度控制 (TIM2_CH2) |
| OUT1/OUT2 | -       | 连接左前电机 |
| OUT3/OUT4 | -       | 连接右前电机 |
| +12V     | -        | 电机电源正极 |
| GND      | GND      | 公共地 |

#### L298N #2 (后轮驱动)
| L298N引脚 | STM32引脚 | 功能说明 |
|----------|----------|---------|
| IN1      | PB0      | 左后轮方向控制1 |
| IN2      | PB1      | 左后轮方向控制2 |
| IN3      | PB6      | 右后轮方向控制1 |
| IN4      | PB7      | 右后轮方向控制2 |
| ENA      | PB8      | 左后轮PWM速度控制 (TIM3_CH3) |
| ENB      | PB9      | 右后轮PWM速度控制 (TIM3_CH4) |
| OUT1/OUT2 | -       | 连接左后电机 |
| OUT3/OUT4 | -       | 连接右后电机 |
| +12V     | -        | 电机电源正极 |
| GND      | GND      | 公共地 |

### PS2手柄接收器连接 (SPI1)
| PS2引脚 | STM32引脚 | 功能说明 |
|--------|----------|---------|
| DAT    | PA6      | MISO (主入从出) |
| CMD    | PA7      | MOSI (主出从入) |
| CLK    | PA5      | SCK (时钟) |
| ATT/CS | PA4      | 片选信号 |
| VCC    | 3.3V     | 电源 |
| GND    | GND      | 地 |

## 软件功能

### 控制方式

#### 数字按键控制
- **↑ (UP)**: 前进
- **↓ (DOWN)**: 后退
- **← (LEFT)**: 左转
- **→ (RIGHT)**: 右转

#### 模拟摇杆控制
- **左摇杆**: 
  - 上下：前进/后退（速度可调）
  - 左右：左转/右转（速度可调）
- 支持死区处理，避免误操作

### PWM控制
- PWM频率：1kHz
- 速度范围：0-1000 (0%-100%)
- 默认速度：700 (70%)

## 编译和烧录

### 前置要求
- ARM GCC工具链
- OpenOCD 或 ST-Link工具
- STM32 HAL库

### 编译
```bash
make
```

### 烧录
使用ST-Link V2或兼容烧录器：
```bash
make flash
```

或使用OpenOCD：
```bash
openocd -f interface/stlink-v2.cfg -f target/stm32f1x.cfg -c "program build/stm32f103c8t6lilcar.elf verify reset exit"
```

## 注意事项

1. **电源隔离**: 确保电机电源和STM32电源分开，但共地
2. **电流保护**: L298N可能需要散热片，大电流时注意温度
3. **信号电平**: PS2接收器使用3.3V电平，与STM32兼容
4. **PWM引脚**: 确保使用支持定时器输出的引脚
5. **手柄配对**: 首次使用需要配对PS2手柄和接收器

## 故障排查

### 电机不转
- 检查L298N电源连接
- 检查电机连接线
- 检查PWM信号输出

### 手柄无响应
- 检查PS2接收器电源和连接
- 检查手柄电池
- 确认手柄已配对

### 小车动作不正确
- 检查电机接线顺序
- 调整代码中的电机方向定义
